Welcome to the Backtest Service

This service allows you to backtest your trading strategies with a historical data.

Features:

  • Download and use Binance's historical market data (including 1s & trades)
  • Iteration over the same-period chunks of klines to implement complex closely-related strategies with low-memory consumption
  • Launch of the Binance Grid bot trading strategy even on trades market data
  • Calculation of metrics
  • Simple UI built using the Pico CSS framework with the result chart construction in Plotly

Get started by signing up, then signing in and navigating to the backtesting tab to start your calculations.

For more information, please visit the Backtest github repository.
